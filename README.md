# Livl Pacman

## Setting up dev env

### Prerequisites

- CMake
- C compiler
- SDL2
- SDL_mixer

### Installation

#### **MacOS**

Using [homebrew](https://brew.sh/index_fr)

```bash
brew install sdl2 sdl_mixer 
```
